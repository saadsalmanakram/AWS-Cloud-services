AWS Amplify Hosting (Front-End Web & Mobile Applications)

>> Best for:
   - Static Websites with CI/CD: Static sites or single-page applications (SPAs) that need continuous integration and deployment.

   - Full-Stack Applications: Hosting front-end and server-side rendered (SSR) web apps built with frameworks like React, Angular, or Vue.js.

   - Mobile Backends: Serving as a backend for mobile applications with integrated hosting and deployment services.

>> Examples: JAMstack websites, front-end frameworks with server-side rendering, mobile application backends.
